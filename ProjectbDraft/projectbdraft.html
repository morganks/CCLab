KineticSong

Inspiration

What is KineticSong?

Kinetic - Song, the name is derived from the instance of motion (kinetic) and combines it with music (Song). This exploration not only showcases the interplay between music and visual art but also offers an engaging perspective on how technology can translate the intangible essence of sound into a tangible, dynamic visual form.

How it works

Inspired by Daniel Sierra's “Oscillate” my project aims to use audio to generate art. This project seeks to translate the intricate elements of music—pace, mood, pitch, and amplitude—into captivating visual representations using p5.js. By analyzing these components of various songs, the project aims to create a synthetic experience, transforming the music into dynamic and unique visual art forms.
